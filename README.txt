# Svenska Helgdagar

Skapa smartare Homey-flöden med svenska helgdagar, arbetsdagar och klämdagar.

Appen ger dig:
- AND-kortet "Är svensk helgdag" (alla eller specifik helgdag).
- AND-kortet "Arbetsdag" med val för att inkludera eller exkludera klämdagar.
- Globala taggar för:
  - dagens helgdagsnamn
  - om idag är arbetsdag
  - om idag är klämdag
  - om idag är officiell svensk helgdag

Exempel:
- Tänd dekorbelysning på helgdagar.
- Kör olika rutiner på arbetsdagar och lediga dagar.
- Filtrera notiser med taggen för helgdag/arbetsdag.

Status uppdateras automatiskt vid midnatt.
